# Lab1-polymorphic-vector
Realisation of a polymorphic vector in C


## Makefile.mk
Собрать всё

make

Запустить программу

make run

Запустить тесты

make test

Очистить

make clean


## Makefile.msvc
Открыть Developer Command Prompt для VS

Собрать всё 

nmake /f Makefile.msvc

Запустить программу

nmake /f Makefile.msvc run

Запустить тесты

nmake /f Makefile.msvc test

Очистить

nmake /f Makefile.msvc clean
